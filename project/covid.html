COVID-19 Resources


This web design and development project was created to allow me to practice with fetching data from APIs using JavaScript while also displaying relevant data that highlights a few statistics regarding COVID-19 around the world, updated live. The website also features a number of resources that are relevant to the pandemic, ranging from official CDC websites to small business resources provided by the U.S. Small Business Administration. Each card shows a peek and description of what the resource has to offer, complemented with an icon.



Along with an embedded interactive map from Esri, a live coronavirus-related news feed is displayed on the right, fetching from a web search API per page refresh. Additionally, a dark mode experience is triggered based on the user's system settings on their device using JavaScript, shown above. The website is currently live at covid19resources.co.
